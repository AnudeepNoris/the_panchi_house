# Panchi House - Restaurant Website

Beautiful, responsive website for The Panchi House restaurant in ECIL, Secunderabad.

## Features

- 🍽️ Premium restaurant branding
- 📱 Fully responsive design
- 🎨 Modern UI with smooth animations
- 📍 Location & contact integration
- 📋 Menu showcase
- 📧 Contact form for reservations
- ⚡ Built with React + Vite
